# Living Doc Templates

Use these sections when updating project state. Do not leave living docs as title-only placeholders.

## Progress

```markdown
### YYYY-MM-DD HH:MM
- Milestone:
- Summary:
- Files changed:
- Commands run:
- Result:
- Next step:
```

## Decision

```markdown
### ADR-000: Title
- Date:
- Status: proposed | accepted | superseded
- Context:
- Decision:
- Alternatives considered:
- Consequences:
```

## Test Entry

```markdown
### YYYY-MM-DD HH:MM
- Command:
- Environment:
- Result: pass | fail
- Output summary:
- Failures:
- Fixes attempted:
```
