# Do Not Crawl / Do Not Copy Policy

Do not crawl, quote, summarize, imitate, or copy other hackathon project pages or submission descriptions.

The official hackathon page may be used only to confirm broad constraints:
- track task
- allowed platform
- judging category
- dates
- required submission format

Do not copy:
- team names
- project names
- project descriptions
- prompts
- screenshots
- README structures
- architecture labels
- code
- UI text
- branding
